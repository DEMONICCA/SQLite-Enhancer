> [!IMPORTANT]
> Features üöÄ:
> 1. `VACUUM`,`REINDEX`,`ANALYZE` To tidy up and optimize the database structure. ‚úîÔ∏è
> 2. `PRAGMA` `journal_mode WAL`, `synchronous`, `cache_size` To improve database performance. ‚úîÔ∏è
> 3. `PRAGMA` `optimize` Additional feature for optimization, only works in the latest version of SQLite. ‚úîÔ∏è
> 4. `PRAGMA` `auto_vacuum FULL` Enables full auto vacuum mode for automatic space management. ‚úîÔ∏è
> 5. The optimization result log will be saved in `/sdcard/Android/DB.log`. ‚úîÔ∏è
